# מערכת רב-לשוניות (i18n) — בשבילנו

מסמך תחזוקה למנגנון השפות. האתר נשאר **סטטי טהור**; `build.py` מייצר עמוד נפרד לכל שפה.

## שפות וכתובות

| שפה | כיוון | כתובת |
|-----|-------|-------|
| עברית (ברירת מחדל) | RTL | `/` (שורש) |
| אנגלית 🇺🇸 | LTR | `/en/…` |
| צרפתית 🇫🇷 | LTR | `/fr/…` |
| ספרדית 🇪🇸 | LTR | `/es/…` |

כל 8 העמודים (index, about, team, how, stories, join, contact, donate) קיימים בכל 4 השפות.

## רכיבים

- **תבניות**: `templates/<page>.html` — HTML עם טוקנים `{{key}}` (יכולים להכיל HTML). אזורי ה-nav/footer הם רק זוג מרקרים `<!-- @nav --> <!-- /@nav -->`.
- **מילונים גלובליים**: `i18n/<lang>.json` — nav, footer, מתג שפה, aria, מטא משותף.
- **מילוני עמוד**: `i18n/fragments/<page>.<lang>.json` — תוכן ספציפי לעמוד, ממוזג מעל הגלובלי. מבנה: `{"<page>": { ... }}`. **חובה שאותם מפתחות יופיעו בכל 4 השפות.**
- **partials**: `partials/nav.html` + `footer.html` — כולל מתג השפה ודגלי SVG (sprite).
- **JS**: `assets/main.js` — שכבת `STR` לפי `<html lang>` ללוקליזציה של הודעות וואטסאפ/מייל/טוסט/תרומה; מתג השפה מטפל בפתיחה/סגירה, סימון פעיל, וזיכרון בחירה (`localStorage`).

## חישוב קישורים אוטומטי (שמירת שפה)

`build.py` מחשב לכל צמד **עמוד × שפה**:
- `url.*` — קישורי ניווט/פוטר, מפנים לאותה שפה אם העמוד קיים בה, אחרת נופלים לעברית.
- `lang_url.*` — יעדי מתג השפה (אותו עמוד בכל שפה, או בית אם לא תורגם).

לכן **אין לתחזק כתובות ידנית** — ברגע שעמוד מתורגם, כל הקישורים אליו בכל השפות מתעדכנים אוטומטית, והשפה הנבחרת "נדבקת" לכל הניווט.

## בנייה מדורגת ובטוחה

עמוד נבנה רק אם התרגומים שלו **מלאים** (כל 4 קבצי ה-fragment קיימים). תבנית שעדיין חסרה תרגום פשוט מדולגת — אפשר להוסיף עמודים אחד-אחד בלי לשבור את ה-build.

```bash
python3 build.py     # אידמפוטנטי
```

## הוספת עמוד מתורגם חדש

1. `templates/<page>.html` — תבנית עם טוקנים `{{<page>.*}}`, נתיבי נכסים `/assets/...`, בלוק hreflang, ומרקרי nav/footer ריקים.
2. `i18n/fragments/<page>.{he,en,fr,es}.json` — אותם מפתחות בכל השפות.
3. `python3 build.py`. (אין צורך לגעת ב-`build.py` או ב-JS.)

## הוספת שפה חדשה

1. `i18n/<lang>.json` (העתקה מ-`en.json` ותרגום), כולל `lang_menu.current_flag`/`current_code`.
2. לכל עמוד: `i18n/fragments/<page>.<lang>.json`.
3. הוסף את הקוד ל-`LANGS` ב-`build.py`.
4. הוסף symbol דגל ב-`partials/nav.html` ופריט במתג (dropdown + drawer), ותא שפה ב-`STR` שב-`main.js`.
5. עדכן את בלוקי ה-hreflang בתבניות ואת `sitemap.xml`.
